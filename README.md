
AI-Driven Adaptive Intrusion Detection System (Zero-Day Focus)
==============================================================

This MSc-level cybersecurity project implements a **novel hybrid anomaly detection framework**
for detecting zero-day network attacks using:

1. Neural Autoencoder (behavioral reconstruction)
2. Isolation Forest (statistical anomaly detection)
3. Dynamic Ensemble Scoring (unique contribution)

Key novelty:
- Dual-model ensemble anomaly scoring
- Zero-day simulation via attack-class exclusion
- Permutation-based explainability
- Designed for MSc admission & thesis extension

Run order:
1. python src/train.py
2. python src/evaluate.py
